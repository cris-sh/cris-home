# Neighborhood - Aplicación Móvil Comunitaria

## Visión general del proyecto

Neighborhood App es una aplicación móvil desarrollada con Expo y React Native para ayudar a las comunidades locales a comunicarse, coordinar eventos, reportar incidencias y mantenerse informadas mediante canales pensados para el vecindario. La app busca fortalecer la confianza, la seguridad y la colaboración entre residentes.

## Propósito y alcance

El objetivo de Neighborhood es crear un espacio digital donde los vecinos puedan compartir avisos, organizar actividades, reportar problemas y acceder a información local relevante en un solo lugar. Reúne las interacciones clave que necesita una comunidad, desde avisos de seguridad y clasificados locales hasta actualizaciones vecinales y servicios cercanos.

## Arquitectura y stack tecnológico

El proyecto está construido sobre un stack móvil moderno con foco en mantenibilidad y preparación para despliegue:

- **Expo + React Native** para desarrollo móvil multiplataforma rápido
- **TypeScript** para tipado fuerte y una estructura escalable
- **JavaScript** para lógica auxiliar y compatibilidad con el ecosistema
- **CSS** para personalización visual puntual
- **Patrones de navegación y estado** para flujos entre pantallas y datos compartidos

## Capacidades principales

- Avisos comunitarios y novedades del vecindario
- Coordinación de eventos y comunicación entre residentes
- Reporte de incidencias y flujos de seguridad local
- Chat e interacción comunitaria
- Soporte para notificaciones push en alertas importantes
- Funciones basadas en ubicación y descubrimiento de recursos cercanos
- Integraciones con servicios externos para información local y contexto comercial

## Distribución y publicación

Expo facilitó las pruebas rápidas, la iteración de funcionalidades móviles y la publicación multiplataforma. También adquirí experiencia práctica gestionando la entrega en Google Play Store, los flujos de release y los aspectos operativos necesarios para mantener una app móvil en producción.
